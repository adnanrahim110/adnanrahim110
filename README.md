## 🚀 What I'm Working On

> **[Magpie POS](https://github.com/adnanrahim110)** — An offline-first, AI-integrated SaaS point-of-sale platform built for modern retail worldwide. Not your typical POS. It ships with intelligent inventory predictions, multi-language support (English + Urdu), JazzCash / EasyPaisa / khata payment integrations, a real-time owner dashboard, and a super admin panel — all in one codebase. Built to work flawlessly even with zero internet.

**Stack:** Next.js · TypeScript · TailwindCSS · Framer Motion · PostgreSQL · Prisma · Better-Auth · Workbox · IndexedDB
